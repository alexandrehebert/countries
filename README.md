# Countries Atlas

Standalone catalog app extracted from the `guesser` project's **View all countries** modal.

## Stack

- `Next.js 16`
- `React 19`
- `TypeScript`
- `Tailwind CSS v4`
- `next-intl`
- `Vitest`

## Features

- Exhaustive searchable country list
- Flag rendering for every entry
- SVG country silhouettes from the world map dataset
- English and French locales

## Run locally

```bash
npm install
npm run dev
```

The app runs on `http://localhost:4103`.

## Quality checks

```bash
npm test
npm run build
```
